# yt2mp3
## Converts a youtube video to a mp3 file with ID3 tags

*Arguments:*
- **"--yt"** - Youtube link
- **"-a"** - Artist
- **"-t"** - Title
- **"-c"** - Cover image path

*Example usage:*
```
python yt2mp3 --yt <youtube link> -a <Artist> -t <title> -c <cover image path>
```

*Installation:*
```
pip install -r requirements.txt
```
